# Misc

Store settings, text-to-pay, shipping, and other utility tools.

## Store

| Tool | Description |
|------|-------------|
| `ghl_get_store_settings` | Get store settings for a location |
| `ghl_update_store_settings` | Update store settings |
| `ghl_list_shipping_zones` | List shipping zones |
| `ghl_create_shipping_zone` | Create a shipping zone |
| `ghl_list_shipping_carriers` | List shipping carriers |
| `ghl_create_shipping_carrier` | Add a shipping carrier |
| `ghl_list_shipping_rates` | List shipping rates |

## Text2Pay

| Tool | Description |
|------|-------------|
| `ghl_create_text2pay` | Create a text-to-pay link |

## Error Management

| Tool | Description |
|------|-------------|
| `ghl_list_errors` | List captured tool errors |
| `ghl_clear_errors` | Clear the error log |
