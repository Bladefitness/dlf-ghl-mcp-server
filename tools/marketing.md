# Marketing

Campaigns, email builder, email schedules, and social media planner.

## Campaigns

| Tool | Description |
|------|-------------|
| `ghl_get_campaigns` | List all campaigns |
| `ghl_get_campaign` | Get a specific campaign |

## Email Builder

| Tool | Description |
|------|-------------|
| `ghl_list_emails` | List email campaigns |
| `ghl_get_email` | Get an email campaign |
| `ghl_create_email` | Create an email campaign |
| `ghl_update_email` | Update an email campaign |
| `ghl_delete_email` | Delete an email campaign |
| `ghl_list_email_campaigns` | List all email campaigns |
| `ghl_get_email_schedule` | Get scheduled email status |
| `ghl_cancel_scheduled_email` | Cancel a scheduled email send |
| `ghl_list_email_templates` | List email templates |
| `ghl_get_email_message` | Get email message content |
| `ghl_delete_email_message` | Delete an email message |

## Social Planner

| Tool | Description |
|------|-------------|
| `ghl_list_social_posts` | List scheduled and published social posts |
| `ghl_get_social_post` | Get a social post |
| `ghl_create_social_post` | Create/schedule a social post |
| `ghl_edit_social_post` | Edit a social post |
| `ghl_delete_social_post` | Delete a social post |
| `ghl_bulk_delete_social_posts` | Delete multiple posts at once |
| `ghl_get_social_accounts` | List connected social accounts |
| `ghl_get_social_account_details` | Get details for a social account |
| `ghl_attach_social_account` | Connect a social account |
| `ghl_detach_social_account` | Disconnect a social account |
| `ghl_get_social_statistics` | Get engagement stats |
| `ghl_get_social_categories` | List post categories |
| `ghl_list_social_tags` | List post tags |
| `ghl_upload_social_csv` | Bulk upload posts via CSV |
| `ghl_get_social_csv_status` | Check CSV upload status |
| `ghl_delete_social_csv` | Delete a CSV upload |
| `ghl_start_facebook_oauth` | Connect Facebook account |
| `ghl_start_instagram_oauth` | Connect Instagram account |
| `ghl_start_google_oauth` | Connect Google account |
| `ghl_start_linkedin_oauth` | Connect LinkedIn account |
| `ghl_start_twitter_oauth` | Connect X/Twitter account |
| `ghl_start_tiktok_oauth` | Connect TikTok account |
| `ghl_start_tiktok_business_oauth` | Connect TikTok Business account |
