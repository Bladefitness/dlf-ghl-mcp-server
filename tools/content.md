# Content

| Tool | Description |
|------|-------------|
| `ghl_list_blogs` | List all blogs |
| `ghl_get_blog` | Get a blog |
| `ghl_create_blog` | Create a blog |
| `ghl_update_blog` | Update a blog |
| `ghl_delete_blog` | Delete a blog |
| `ghl_list_blog_posts` | List posts in a blog |
| `ghl_get_blog_post` | Get a blog post |
| `ghl_create_blog_post` | Create a blog post |
| `ghl_update_blog_post` | Update a blog post |
| `ghl_delete_blog_post` | Delete a blog post |
| `ghl_list_blog_authors` | List blog authors |
| `ghl_list_blog_categories` | List blog categories |
| `ghl_check_url_slug` | Check if a URL slug is available |
