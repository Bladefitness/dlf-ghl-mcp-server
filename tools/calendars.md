# Calendars

Calendar management, scheduling, appointments, and availability.

## Tools

| Tool | Description |
|------|-------------|
| `ghl_list_calendars` | List all calendars in a location |
| `ghl_get_calendar` | Get a calendar by ID |
| `ghl_create_calendar` | Create a new calendar |
| `ghl_update_calendar` | Update calendar settings |
| `ghl_delete_calendar` | Delete a calendar |
| `ghl_list_calendar_groups` | List calendar groups |
| `ghl_get_calendar_group` | Get a calendar group |
| `ghl_create_calendar_group` | Create a calendar group |
| `ghl_update_calendar_group` | Update a calendar group |
| `ghl_delete_calendar_group` | Delete a calendar group |
| `ghl_update_calendar_group_status` | Enable or disable a calendar group |
| `ghl_validate_group_slug` | Check if a group URL slug is available |
| `ghl_list_calendar_resources` | List rooms and equipment |
| `ghl_get_calendar_resource` | Get a calendar resource |
| `ghl_create_calendar_resource` | Create a calendar resource |
| `ghl_update_calendar_resource` | Update a calendar resource |
| `ghl_delete_calendar_resource` | Delete a calendar resource |
| `ghl_list_calendar_events` | List all calendar events (appointments + blocked slots) |
| `ghl_list_blocked_slots` | List blocked time slots |
| `ghl_create_blocked_slot` | Block a time slot |
| `ghl_update_blocked_slot` | Update a blocked slot |
| `ghl_delete_blocked_slot` | Remove a blocked slot |
| `ghl_get_appointment` | Get an appointment by ID |
| `ghl_create_appointment` | Book a new appointment |
| `ghl_update_appointment` | Update an appointment |
| `ghl_delete_appointment` | Cancel/delete an appointment |
| `ghl_get_calendar_free_slots` | Get available booking slots for a date range |
| `ghl_get_appointment_notes` | Get notes on an appointment |
| `ghl_create_appointment_note` | Add a note to an appointment |
| `ghl_update_appointment_note` | Edit an appointment note |
| `ghl_delete_appointment_note` | Delete an appointment note |
| `ghl_list_calendar_notifications` | List calendar notifications |
| `ghl_create_calendar_notification` | Create a calendar notification |
| `ghl_update_calendar_notification` | Update a notification |
| `ghl_delete_calendar_notification` | Delete a notification |
