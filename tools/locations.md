# Locations & Settings

Location management, custom fields, custom values, tags, and templates.

## Location Management

| Tool | Description |
|------|-------------|
| `ghl_get_location` | Get location details |
| `ghl_search_locations` | Search locations |
| `ghl_create_location` | Create a new location |
| `ghl_update_location` | Update location settings |
| `ghl_delete_location` | Delete a location |
| `ghl_get_location_timezones` | List available timezones |
| `ghl_get_location_business` | Get the business profile for a location |
| `ghl_update_location_business` | Update the business profile |
| `ghl_get_business_profile` | Get business profile details |
| `ghl_update_business_profile` | Update business profile |
