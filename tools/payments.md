# Invoices & Payments

Invoices, estimates, orders, subscriptions, products, and payment processing.

## Invoices

| Tool | Description |
|------|-------------|
| `ghl_list_invoices` | List invoices with filters |
| `ghl_get_invoice` | Get an invoice by ID |
| `ghl_create_invoice` | Create a new invoice |
| `ghl_update_invoice` | Update an invoice |
| `ghl_delete_invoice` | Delete an invoice |
| `ghl_send_invoice` | Send an invoice to the contact |
| `ghl_record_invoice_payment` | Record a manual payment |
| `ghl_void_invoice` | Void an invoice |
| `ghl_generate_invoice_number` | Generate the next invoice number |
| `ghl_list_invoice_templates` | List invoice templates |
| `ghl_get_invoice_template` | Get a template |
| `ghl_create_invoice_template` | Create a template |
| `ghl_update_invoice_template` | Update a template |
| `ghl_delete_invoice_template` | Delete a template |
| `ghl_list_invoice_schedules` | List recurring invoice schedules |
| `ghl_get_invoice_schedule` | Get a schedule |
| `ghl_create_invoice_schedule` | Create a recurring schedule |
| `ghl_update_invoice_schedule` | Update a schedule |
| `ghl_delete_invoice_schedule` | Delete a schedule |
| `ghl_schedule_invoice` | Schedule an invoice to send later |
| `ghl_cancel_scheduled_invoice` | Cancel a scheduled send |
| `ghl_manage_schedule_auto_payment` | Enable/disable auto-payment on a schedule |
